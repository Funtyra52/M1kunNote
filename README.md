# 📝 M1kunNote — Минималистичный Markdown-Блокнот

*Список горячих клавиш - как и во всех блокнотах!*

**M1kunNote** — это современное, легкое и безопасное десктопное приложение для работы с заметками в формате Markdown. Разработано на базе связки **Electron + Vite + TypeScript** без использования тяжелых веб-фреймворков, что обеспечивает максимальное быстродействие и отзывчивость интерфейса.

---

## ✨ Основные возможности

*   **Split View (Редактирование в реальном времени)**: Три режима отображения (только редактор, разделенный экран и только превью) позволяют сразу видеть результат форматирования.
*   **Автосохранение**: Встроенный умный дебаунс-таймер (200 мс для рендера, 1 секунда для сохранения) автоматически записывает ваши изменения на диск во время ввода.
*   **Подсветка синтаксиса (Syntax Highlighting)**: Интегрированная библиотека `highlight.js` с темой GitHub Dark подсвечивает блоки кода (JS, Python, TS, HTML, CSS и др.) прямо в окне предпросмотра.
*   **Безопасность (XSS-фильтрация)**: Весь генерируемый HTML-код проходит через очистку `DOMPurify`, предотвращая возможность удаленного выполнения кода (RCE) через вредоносные скрипты в файлах `.md`.
*   **Удобный файловый менеджер**: Быстрое переключение между файлами с помощью вкладок (табов), создание и удаление папок/файлов в проводнике и поиск.
*   **Офлайн-установщик (NSIS)**: Удобный мастер установки с возможностью выбора директории и ярлыков, оформленный кастомной иконкой.

---

## 🛠️ Стек технологий

*   **Core**: Electron 39, Vite 7, TypeScript 5 (Vanilla TS в рендерере для легковесности)
*   **Markdown Parsing**: `marked` (поддержка GFM — GitHub Flavored Markdown)
*   **Security**: `dompurify`
*   **Highlighting**: `highlight.js` (стиль GitHub Dark)
*   **Installer**: `electron-builder` + NSIS ( assisted installer )

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск в режиме разработки (Hot Reload)
```bash
npm run dev
```

### 3. Сборка приложения

```bash
# Сборка установщика для Windows (.exe)
npm run build:win

# Сборка для macOS (.dmg)
npm run build:mac

# Сборка для Linux (.deb / .AppImage)
npm run build:linux
```

Готовые файлы установщиков будут находиться в папке `dist/`.

## 🔒 Безопасность и архитектура

Приложение использует разделение процессов Electron:
*   **Main-процесс**: Управляет жизненным циклом окон и предоставляет безопасные операции чтения/записи файлов через защищенный мост `preload`.
*   **Preload-скрипт**: Экспонирует в контекст рендерера только строго необходимые IPC-события через `contextBridge.exposeInMainWorld`, исключая прямой доступ рендерера к Node.js.
*   **Renderer-процесс**: Отвечает за отрисовку интерфейса. Все данные перед выводом на экран очищаются от потенциально опасных тегов и скриптов.
